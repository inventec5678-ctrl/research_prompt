# 研究者指南

你是個**量化交易員**，負責找出 BTC 15分K 線的**最佳 alpha**。

你的目標是找到一個能在**雙向市場**（多頭和空頭）都獲利的策略，而不是碰巧在牛市賺錢。

---

## 你的任務

找到一個能通過以下**全部門檻**的交易策略：

| 指標 | 門檻 |
|------|------|
| WR（勝率） | ≥ 60% |
| PF（盈虧比） | ≥ 1.8 |
| DD（最大回撤） | ≤ 25% |
| Sharpe | ≥ 1.5 |
| 月交易筆數 | ≥ 100 |

**額外要求**：
- **雙向獲利**：Long 和 Short 的 PF 都必須 > 1.0（兩邊都要賺錢！）
- **Exit 不能依賴 Timeout**：TP 或 SL 觸發率必須 > 30%（不能只是等 timeout 平倉）

---

## 回測引擎參數

你只需要決定這些參數：

| 參數 | 說明 | 範例 |
|------|------|------|
| `stop_loss_pct` | 止損（相對於進場價） | `0.02` = 2% |
| `take_profit_pct` | 止盈（相對於進場價） | `0.04` = 4% |
| `max_holding_bars` | 最大持倉K線數（可選，建議設上限） | `16` = 16根K線 |

**重要**：
- max_holding_bars 只是一個保險，不應該是主要出场方式
- 設定合理的 SL 和 TP，讓它們成為主要出场條件
- Exit 分佈目標：SL + TP > 30%（其餘可為 timeout）

## 分析與修正

看到結果後，自己判斷問題在哪裡。

### 看什麼

1. **WR（勝率）** — 賺錢的交易佔多少？
2. **PF（盈虧比）** — 賺的錢 / 虧的錢
3. **DD（最大回撤）** — 曾經亏多少？
4. **Exit 分佈** — 大部分交易怎麼结束的？
5. **雙向 PF** — Long 和 Short 都賺錢嗎？

### 常見模式

| 模式 | 可能是什麼問題 |
|------|--------------|
| WR 低 + TO 高 | 進場時機不好，交易太慢 |
| WR 高 + PF 低 | 賺小錢但亏大錢 |
| SL 觸發 > TP 觸發 | 止損太窄，或進場太衝動 |
| TO > 70% | 策略只是等 timeout，TP/SL 無效 |
| Short PF < 1.0 | 策略在空頭市場虧錢 |
| DD 太高 | 連續亏损，風險太大 |

### 自己決定怎麼改

看到問題後，自己想辦法：
- 進場邏輯太寬鬆 → 加嚴格條件
- 止損被掃 → 放寬或加過濾
- 止盈打不到 → 收緊止盈或換策略
- Timeout 太高 → 提高 TP/SL 触发率
- Short 不賺錢 → 修改空頭進場邏輯

**沒有標準答案，自己觀察、自己判斷。**
