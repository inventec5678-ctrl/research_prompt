## 分析與修正

看到結果後，自己判斷問題在哪裡。

### 看什麼

1. **WR（勝率）** — 賺錢的交易佔多少？
2. **PF（盈虧比）** — 賺的錢 / 虧的錢
3. **DD（最大回撤）** — 曾經亏多少？
4. **Exit 分佈** — 大部分交易怎麼结束的？

### 常見模式

| 模式 | 可能是什麼問題 |
|------|--------------|
| WR 低 + TO 高 | 進場時機不好，交易太慢 |
| WR 高 + PF 低 | 賺小錢但亏大錢 |
| SL 太高 | 止損太窄，或進場太衝動 |
| TP 太低 | 止盈太嚴格，價格到不了 |
| DD 太高 | 連續亏损，風險太大 |

### 自己決定怎麼改

看到問題後，自己想辦法：
- 進場邏輯太寬鬆 → 加嚴格條件
- 止損被掃 → 放寬或加過濾
- 止盈打不到 → 收緊止盈或換策略
- 什麼都沒發生 → 表示市場沒給機會

**沒有標準答案，自己觀察、自己判斷。**
