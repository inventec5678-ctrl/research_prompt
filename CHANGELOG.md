# 研究迭代日誌 (Research Iteration Log)

## 專案目標

找出 BTC 15分K 交易策略，通過以下門檻：
- WR（勝率） ≥ 60%
- PF（盈虧比） ≥ 1.8
- DD（最大回撤） ≤ 25%
- Sharpe ≥ 1.5
- 月交易筆數 ≥ 100

## 教訓 (Lessons Learned)

1. **Timeout 不是有效的 Exit**：策略必須靠 TP 或 SL 觸發平倉
2. **牛市 beta ≠ 策略 alpha**：需要雙向市場驗證
3. **高 Sharpe 可能只是運氣**：需要多個配置和多段數據驗證
4. **每次測試都要儲存**：方便未來檢視和避免重複
